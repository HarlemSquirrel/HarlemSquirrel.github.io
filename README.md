## Dependencies

- [Ruby](https://www.ruby-lang.org/)

## Run locally

```sh
# Clone the respository
git clone git@github.com:HarlemSquirrel/HarlemSquirrel.github.io.git

# Install Ruby gems
bundle install

# Run the development server
jekyll serve
```
